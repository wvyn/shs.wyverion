Cross-Platform Play (Java & Bedrock Together)

This setup allows both Java and Bedrock players to play together on the same world. However, they must connect using different ports.

  • Java Edition — No port needed: 
    vanilla.staffordhighschool.net
  • Bedrock Edition — Use this port for cross-play support: 
    vanilla.staffordhighschool.net:19134

Important Notes:

  • Both versions will join the same world with shared progress.
  • Ensure your game version matches the server’s (typically the latest stable release).
  • If you’re on Bedrock and can’t connect, make sure to use the correct port (19134).
